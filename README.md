# Figure Reproduction: Biogas Production

## Project Objective
This project aims to reproduce a scientific figure from a biogas production study using **R** and **ggplot2**. It demonstrates a clean data science workflow, from raw data to a publication-ready plot.

## Project Structure
- `data/`: Contains the raw dataset in CSV format.
- `scripts/`: Contains the R code for data processing and visualization.
- `figures/`: Contains the final generated plots.

## How to Run
1. Open the `.Rproj` file in RStudio.
2. Run the script located in `scripts/plot_figure.R`.
